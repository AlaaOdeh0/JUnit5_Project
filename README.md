# JUnit5_Project

This project demonstrates unit testing using **JUnit 5** in Java. It includes test cases for:
- **Calculator**
- **Product**
- **User Service**
- **Recipe Book**


## 📂 Project Structure

src/ │── main.najah.code/ # Application Code │
                         ├── Calculator.java │ 
                         ├── Product.java │ 
                         ├── RecipeBook.java │ 
                         ├── UserService.java │ 
     │── main.najah.test/ # Test Cases │ 
                         ├── CalculatorTest.java │ 
                         ├── ProductTest.java │ 
                         ├── RecipeBookTest.java │ 
                         ├── UserServiceSimpleTest.java │ 
                         ├──TestSuite.java |


## 🛠️ How to Run Tests
1. Open **Eclipse**.
2. Right-click on `TestSuite.java` → **Run As → JUnit Test**.
3. View the **Test Results & Coverage Report**.

## 🚀 Technologies Used
- **Java 24**
- **JUnit 5**
- **Eclipse IDE**
- **JaCoCo** (for test coverage)
